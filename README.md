## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de criar um jogo onde um grupo de participantes pode **cronometrar quanto tempo cada um leva para realizar uma tarefa**, seja em jogos ou em situações do dia a dia.

A cada cronometragem, um **ranking é atualizado automaticamente**, exibindo os tempos e os nomes dos participantes, além de destacar os três melhores com medalhas 🥇🥈🥉.

> 💡 É um projeto simples, divertido e útil para desafios entre amigos!

## 💻 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/andre55z/JogoComCronometro.git
```

2. Acesse a pasta:
```bash
cd JogoComCronometro
```

3. Execute o arquivo `index.html` no seu navegador.

---

## ✨ Funcionalidades

- ⏱️ Cronômetro com segundos e milissegundos  
- 👥 Cadastro de participantes com verificação de duplicados  
- 🏆 Ranking automático com medalhas para os 3 primeiros colocados  
- 🔁 Reset do cronômetro com botão dedicado  
- 🎨 Feedback visual nos botões (vermelho, verde)
